# Smart Assistant Chatbot

A simple Generative AI assistant built using Python and Ollama.

## Features
- Multi-turn conversation
- Summarize text
- Explain code
- Chat logging

## Tech Stack
- Python
- Ollama
- Llama3 Model

## How to Run

1. Install Ollama
2. Run the model:
ollama run llama3

3. Run the assistant:
python ai_assistant.py
